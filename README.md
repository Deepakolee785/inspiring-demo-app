# Technology used:
1. React
2. Redux
3. Material-ui

# Installation
## clone
git clone https://github.com/Deepakolee785/inspiring-demo-app.git

## dependencies install and start app
cd inspiring-demo-app <br>
npm run server-install <br>
npm install <br>
npm run dev <br>
